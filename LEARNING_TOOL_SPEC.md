# Learning Tool Specification

## Ziel
Jede*r Teilnehmende* baut ein eigenes Learning Tool und fuehrt die Daten spaeter in ein gemeinsames Dashboard zusammen.

## Phase 1: Individuelles Tool
Das Tool erfasst:
- Lernstand
- Lernziele
- agile Lernplanung

## Phase 2: Gemeinsames Tool
Das Dashboard zeigt:
- Gemeinsamkeiten
- Abweichungen
- Vorschlaege fuer Lerngruppen (2+ Teilnehmende*)

## Mindestfunktionen
- Persistenz
- Visualisierung
- Export
- Basis-API fuer Aggregation

## Sprache
User Stories und Begleittexte sind genderneutral mit * formuliert.
